# Ansible Role: shell todo.txt

Ansible role for configuring [todo.txt](http://todotxt.org/) in the terminal

## Usage

### List tasks
```$ t```

### Add tasks
```$ t add "some task```

### Bash completion
```$ t <tab>```

None
